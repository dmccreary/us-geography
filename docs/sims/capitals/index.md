# State Capitals Quiz

An interactive MicroSim to help 5th graders learn all 50 US state capitals through two engaging modes: Study Mode for learning and Quiz Mode for testing knowledge.

## About This MicroSim

This educational tool is designed with 5th graders in mind, using pedagogical principles that reduce frustration and encourage learning:

- **Study Mode**: Browse through all 50 states at your own pace, seeing each state highlighted on the map with its capital
- **Quiz Mode**: Test your knowledge with multiple-choice questions (3, 5, or 10 questions)
- **Visual Learning**: States are highlighted on an interactive map as you study or answer questions
- **Positive Reinforcement**: Celebration animations and encouraging messages keep learners motivated
- **Growth Mindset**: Wrong answers show the correct answer with supportive feedback

## How to Use

### Study Mode
1. Click "Study Mode" on the start screen
2. Use "Previous" and "Next" buttons to browse states
3. Click "Shuffle States" to randomize the order
4. The map zooms to each state as you study

### Quiz Mode
1. Click "Quiz Mode" on the start screen
2. Select your quiz length: 3, 5, or 10 questions
3. Click "Start Quiz!" to begin
4. Click the correct capital from four choices
5. Watch your score and collect stars for correct answers!
