# About This Book

This intelligent textbook teaches US Geography using interactive simulations, learning graphs, and modern web-based learning tools.

## Target Audience

Students and educators interested in learning about the geography of the United States.

## How to Use This Book

1. **Navigation** - Use the left sidebar to navigate between chapters
2. **Search** - Use the search bar at the top to find specific topics
3. **MicroSims** - Interactive simulations help visualize geographic concepts
4. **Learning Graph** - Explore concept dependencies and learning pathways
